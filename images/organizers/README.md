# Organizer Photos

This directory contains photos of the shared task organizers.

## Adding Photos

To add organizer photos:

1. Add square photos (recommended: 300x300px or larger) to this directory
2. Name them according to the pattern used in index.html:
   - `pranav-gupta.jpg`
   - `niranjan-kumar.jpg`
   - `balaji-nagarajan.jpg`
   - Add more as needed

3. Supported formats: JPG, PNG, or WebP
4. Recommended size: 300x300px to 500x500px (square aspect ratio)
5. File size: Keep under 200KB for optimal loading

## Current Photos

- `placeholder.jpg` - Default placeholder image (SVG)

## Fallback

If a photo is not found, the placeholder image will be displayed automatically.
